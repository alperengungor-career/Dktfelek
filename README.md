# Dktfelek - Eğitici Alfabe Oyunu

Dil ve Konuşma Terapistlerinin (DKT) seanslarında çocuk danışanlarıyla kullanabileceği, sesbilgisel farkındalığı artırmaya yönelik web tabanlı interaktif bir eğitim materyalidir. Uygulama, çocukların oyun oynarken harfleri tanımasını ve o harflerle ilgili dil görevlerini yerine getirmesini hedefler.

## Öne Çıkan Özellikler
* **Kapsamlı İçerik:** Türk alfabesindeki her harf için özel olarak kurgulanmış eğitici görevler ve bilmeceler.
* **Ses Sentezi (Speech Synthesis API):** Görevlerin sesli okunma özelliği sayesinde okuma bilmeyen çocuklar için erişilebilirlik desteği.
* **PWA Desteği:** Uygulamanın mobil cihazlara kurulabilmesi ve çevrimdışı kullanım imkanı (Progressive Web App).
* **İlerleme Takibi:** localStorage teknolojisi kullanılarak tamamlanan harflerin cihaz hafızasında saklanması.
* **Duyarlı Tasarım:** Masaüstü ve mobil cihazlar için tam uyumlu (Responsive) kullanıcı arayüzü.

## Kullanılan Teknolojiler
* **Frontend:** HTML5, CSS3
* **Mantık Katmanı:** Vanilla JavaScript (DOM Manipülasyonu ve Oyun Algoritmaları)
* **API Uygulamaları:** Web Speech API (Speech Synthesis)
* **Konfigürasyon:** Web App Manifest (PWA)
* **Dağıtım:** Vercel (CI/CD Pipeline via GitHub)
